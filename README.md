# 🌦️ WeatherApp

[![Flutter](https://img.shields.io/badge/Built%20with-Flutter-02569B.svg)](https://flutter.dev)  
A simple yet powerful **Weather Application** built with Flutter that lets users fetch and display real-time weather information.  
Get forecasts by city or current location — perfect for learning API integration and creating real-world mobile apps.

---

## 📌 About

**WeatherApp** is a cross-platform mobile application built using **Flutter** and **Dart**. It integrates with a weather API (such as OpenWeatherMap) to fetch current weather data and display it in an intuitive user interface.



---

## 🌟 Features

✨ Fetch current weather using city search  
📍 Get weather by current device location *(if enabled)*  
🌡️ Shows temperature, humidity, wind & basic weather details  
☀️ Clean, responsive UI across devices  
⚡ Easy to extend with forecasts, maps, or themes

---

## 🛠 Tech Stack

| Technology | Purpose |
|------------|---------|
| **Flutter** | UI & cross-platform framework |
| **Dart** | Main programming language |
| **Weather API** | Current weather data |
| **HTTP / Dio** | API networking *(optional)* |
| **Location Services** | To get user’s location *(optional)* |
